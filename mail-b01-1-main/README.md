## 作業說明
1. 註冊系統： 滿足下列流程的系統，細節由同學自行發揮，可於報告時說明特色：
   申請帳號：填寫基本資料( 含 郵件位址)，送出申請。
   系統寄確認函，信函中包含可以啟用帳號的連結。
   申請者收到確認信函時，點選啟用連結即可開通帳號 (需設計開通碼，防止攻擊)。
   開通後始可登入。登入後始可編輯自己的基本資料。
   設計登出功能。
2. 寄送通知系統:
   系統管理員打開系統，輸入要寄送的信件內容，系統會將通知信件寄送給名單上的使用者。
   使用者的資料及郵件如何取得，由同學們自行設計(也可使用第1題中的使用者名單)。demo 時，寄送名單上需有我的測試郵件。
